⚙️ Setup sekali — paste security rule ini ke Firebase Console (Build → Firestore → Rules → Publish)

Tambahkan di dalam match /databases/{database}/documents { … }, lalu Publish. Tanpa ini, Generate gagal (permission denied — sudah aku tes: create = 403).

match /invoices/{id} {
  allow read: if true;
  allow create: if request.resource.data.status == 'sent'
                && request.resource.data.signature_png == null;
  allow update: if resource.data.signature_png == null
                && request.resource.data.diff(resource.data).affectedKeys()
                     .hasOnly(['signature_png','signed_at','signed_meta','status'])
                && request.resource.data.status == 'signed';
  allow delete: if false;
}
