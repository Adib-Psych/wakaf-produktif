Personal Learning Garden — Adib's SOP Study

Kurikulum modular 19 topik SOP perawatan Robusta Kebun Wakaf Ngantang, dirancang untuk pendalaman teoretis dengan pendekatan praktis. Setiap modul memuat narasi, visual dari sumber terbuka (open resources), step-by-step praktik, dan refleksi diri. Catatan & progres tersimpan otomatis di perangkat Anda.

📖 Filosofi Belajar
🎯 Format Hybrid · Personal Learning Garden

Setiap modul mengikuti alur: Story Intro → Konteks Ngantang → Teori Dasar (dengan analogi psikologi untuk schema building) → Visual Tour (foto open-source) → Deep Dive (komparasi 4-arah + video referensi) → Praktik Step-by-Step (checklist tersimpan) → Refleksi (3 prompt, catatan tersimpan otomatis).

📷 Sumber Media Terbuka

Foto, video, dan infografis di setiap modul diambil dari sumber terbuka (Wikimedia Commons, Unsplash, YouTube CC-licensed, FAO image library, jurnal Open Access) dengan atribusi lengkap. Strategi ini memungkinkan modul live secepatnya, dan akan diganti bertahap dengan media produksi sendiri seiring waktu.

💾 Privacy & Storage

Catatan refleksi dan progres checklist tersimpan secara lokal di perangkat ini saja (browser localStorage) — tidak dikirim ke server mana pun. Untuk akses dari perangkat lain, ekspor catatan secara manual.
